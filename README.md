# Pizza Marinara for HaC

This mod adds pizza marinara for Heat And Climate.

## Requirement

- Minecraft 1.12.2
- Minecraft Forge 1.12.2-14.23.3.2655+
- Heat And Climate Lib 1.12.2-3.3.1+
- Heat And Climate Mod 1.12.2-3.3.3+
